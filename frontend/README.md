# Voice CRM Frontend

Next.js app for the Slack + CRM hybrid cockpit described in the UX plan and interaction specs.

## Running locally
1. `npm install`
2. `npm run dev`
3. Open `http://localhost:3000`

The UI mirrors the three-pane Agent Desktop, Pipeline board, Conversation detail, and Supervisor queue views, using static data to demonstrate the layout and interaction affordances.
